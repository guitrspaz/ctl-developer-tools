    updateSpecialAccessToQuiz

    public void updateSpecialAccessToQuiz(numeric studentID, numeric quizID, numeric courseOfferingID, numeric granterID, string startDate, string endDate, [numeric specialTimeToComplete='0'], [numeric numTimesCanAccess='0'], [boolean lockExistingAnswers='false'], [numeric numAttemptsAllowed='0'])

    Updates the record for a student to have special access to a specific quiz in a specific course.

    Parameters:
    studentID - The integer PKEY of the student whose record we want.
    quizID - The integer PKEY of the quiz we want to query on.
    courseOfferingID - The integer PKEY of the courseOfferingID to assign to.
    granterID - The integer PKEY of the user who granted special access.
    startDate - The date that special access starts on for this student, empty if we're just updating the specialTimeToComplete value.
    endDate - The date that special access ends for this student, empty if we're just updating the specialTimeToComplete value.
    specialTimeToComplete - The numeric value, in minutes, that this student has to complete this quiz.
    numTimesCanAccess - Numeric indicating the number of times the student can access the quiz -- as in more than the normal one time.
    lockExistingAnswers - Boolean indicating if the student should not be able to change existing answers, applicable only if student is granted additional access to the quiz.
    numAttemptsAllowed - Numeric indicating the maximum number of full attempts the student can have on this quiz.
